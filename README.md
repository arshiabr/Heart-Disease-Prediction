# Heart-Disease-Prediction
Classify whether a patient has heart disease using the UCI Heart Disease dataset

## Dataset Description
You will use the UCI Heart Disease dataset1 which can be directly loaded using the ucimlrepo Python package.
https://pypi.org/project/ucimlrepo/

This dataset contains several clinical features, such as:
• Age, Sex, Chest Pain Type, Resting Blood Pressure, Serum Cholesterol,
• Fasting Blood Sugar, Resting ECG, Maximum Heart Rate, Exercise Induced Angina,
• ST Depression, Slope of ST Segment, Number of Major Vessels, and Thalassemia.
The target variable represents the presence (1) or absence (0) of heart disease.
